## How to refresh

1. `cargo run -p sbol-ontology --bin update-sbol-owl3-fixture` — re-pin the OWL against the current `main` of `SynBioDex/sbol-owl3`.
2. `cargo run -p sbol3 --bin generate-sbol-owl3-conformance-report` — regenerate this file.
3. `cargo test -p sbol3 --test sbol_owl3_conformance --test sbol_owl3_conformance_report` — confirm the assertions and the freshness gate pass.
4. Commit the pin, manifest, this report, and any allowlist changes in one commit so the trail is auditable.
