# sbol3-12114: Variable Feature derivation roles inherit

**Status:** Partial  •  **Spec section:** §6.5  •  **PolicyOptions knob:** `derivation_provenance`

## Spec wording

> A derived Feature from a variable template SHOULD inherit role properties from the variable Feature.

## Ambiguity

Same shape as sbol3-12108.

## Decision

Same shape as sbol3-12108: warn on non-superset roles.

## Rationale

See sbol3-12108.
