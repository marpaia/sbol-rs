# sbol3-12303: Faithful Built Component

**Status:** Deferred  •  **Spec section:** §6.6  •  **PolicyOptions knob:** `(none; coverage-only)`

## Spec wording

> An Implementation's `wasBuiltFrom` Component SHOULD faithfully represent the actual built artifact.

## Ambiguity

▲ rule. Faithfulness is a wet-lab assertion; no algorithm can verify it.

## Decision

Coverage records `MachineUncheckable`. No `PolicyOptions` knob.

## Rationale

Honoring the spec's ▲ guidance: this is producer-asserted, not validator-decidable.

## Examples

### Conservative (default)
Matches today's validator behavior: the diagnostic shape and severity are exactly what the current test suite expects.

### Strict
(promotes the warning to an error, or activates a check that is currently suppressed.)

### Lenient
(drops the check; the rule is recorded in coverage as suppressed.)
