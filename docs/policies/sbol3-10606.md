# sbol3-10606: DNA/RNA topology completeness

**Status:** Partial  •  **Spec section:** §6.4  •  **PolicyOptions knob:** `topology_completeness`

## Spec wording

> A Component that includes a DNA or RNA type property MUST also have a topology property that uses a term from Table 3 if the component's topology is known.

## Ambiguity

"If known" is a producer-side assertion. The validator cannot tell whether topology is known by the producer; it can only check that a topology is present.

## Decision

Default `Conservative`: missing topology is undecided. `RequireKnownForNucleicAcids` (a `TopologyCompleteness` knob, already shipped) treats DNA/RNA without a known SO topology term as error.

## Rationale

Most production documents are partial drafts; refusing them by default would block iterative design. Opt-in for CI gates.
